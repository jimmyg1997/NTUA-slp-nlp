# Introduction in language representations
## Project Description
### PART A : Spell Checker
In the preperation lab, we implement a **spell checker** using FST (finite state machines) desgined with the 
[OpenFST Library](http://www.openfst.org/twiki/bin/view/FST/WebHome).

### PART B : Sentiment Analysis
In the lab, we implement a **sentiment classifier** that categorizes IMDB reviews for movies to positive or negative (**binary classification**)
